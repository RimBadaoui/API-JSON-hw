Facts about Cats

More About This API

  • This is an API that returns random facts about cats and takes a perameter for the number of facts to return.
  • The documentation for this API is available at this link
  • I used the perameter with the key "limit", which tells the API how many facts to return. I set this using the value inputted by the user.
  • This API would be useful as a widget on a pet shelter website so that clients can learn a bit about cats before adopting one.
